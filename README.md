# Iacono Clima

Sito di **Iacono Climatizzazione S.r.l.** — Siracusa.
Single-page con catalogo prodotti su `/prodotti`.

## Stack

React 19 + Vite 8 + TypeScript 5 + Tailwind CSS v4 +
Framer Motion + GSAP/ScrollTrigger + Lenis + Leaflet.

## Comandi

```bash
npm install
npm run dev        # dev server localhost:5173
npm run build      # tsc -b && vite build
npm run preview    # serve dist/
npm run lint
npm run typecheck
```

Per la documentazione tecnica completa (design system, struttura, convenzioni) leggi `CLAUDE.md`.
